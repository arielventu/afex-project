# App "Mis videos favoritos"

### Proyecto desarrollado con las siguientes Tecnologías y Frameworks:
### Frontend:
ReactJS, CSS puro y Bootstrap

### Backend:
Serverless, NodeJS en AWS Lambda y AWS Dynamodb para la persistencia de datos
